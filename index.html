Favourites

1. The HSL slider implements the Hue-saturation-lightness model (HSL).

2. To add a color into favourites click on the color in the circular dial.

3. The colors in the dial and in the favourites columns will cease to exist if you refresh the page as the data is stored in the variables within the script. Refreshing the page sets the variable to empty set.

4. The hex code for the current color is generated in the hex color box.
